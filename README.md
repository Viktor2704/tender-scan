# 📋 ТендерСкан

Приложение для поиска и парсинга тендеров с площадок Bidzaar, B2B-Center и Фабрикант.

## Возможности

- **Сводка** — дашборд с аналитикой по тендерам
- **Парсер** — имитация парсинга всех страниц площадок с логами
- **Тендеры** — таблица с фильтрами, поиском, сортировкой и оценкой (Советую / На проверку / Против)
- **Загрузка XLSX** — импорт данных из файла СВОДКА_ВСЕ_ТЕНДЕРЫ.xlsx
- **Папки** — управление структурой 01_Входящие → 05_Готовые_пакеты

## Быстрый старт

```bash
# 1. Клонировать репозиторий
git clone https://github.com/ТВОЙ_ЛОГИН/tender-scan.git
cd tender-scan

# 2. Установить зависимости
npm install

# 3. Запустить локально
npm run dev
```

Откроется на http://localhost:3000

## Деплой на Vercel (бесплатно)

1. Зайди на [vercel.com](https://vercel.com) и войди через GitHub
2. Нажми **New Project** → выбери репозиторий `tender-scan`
3. Vercel автоматически определит Vite — просто нажми **Deploy**
4. Через 30 секунд получишь рабочую ссылку

## Деплой на Netlify (бесплатно)

1. Зайди на [netlify.com](https://netlify.com) и войди через GitHub
2. Нажми **Add new site** → **Import an existing project** → выбери репозиторий
3. Build command: `npm run build`
4. Publish directory: `dist`
5. Нажми **Deploy**

## Формат XLSX для импорта

Файл должен содержать столбцы (названия можно на русском или английском):

| Столбец | Пример |
|---------|--------|
| Номер | BZ-2026-4821 |
| Название | Модернизация видеонаблюдения |
| Площадка | bidzaar / b2b / fabrikant |
| Заказчик | ПАО «Ростелеком» |
| Регион | Москва |
| Цена | 4850000 |
| Дедлайн | 2026-03-28 |
| Оценка | Советую / На проверку / Против |
| Комментарий | Свободный текст |
